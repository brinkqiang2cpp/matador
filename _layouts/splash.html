matador

matador is an object relational mapping framework written in C++. It aims to encapsulate all database backend communication. You dont have to deal with database backends or sql statements neither with mapping of data types or serialization of objects. It comes with relation handling out of the box, brings a unique container for, a fluent sql query interface. On top all features are combined to a ORM layer.
