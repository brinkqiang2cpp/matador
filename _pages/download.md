**Sources**

Get the sources from GitHub and enter the created directory:

```bash
$ git clone https://github.com/zussel/oos.git
$ cd oos
```

Or get the pre-packaged sources:

[zip](#link){: .btn .btn--success} [tar.gz](#link){: .btn .btn--success}
